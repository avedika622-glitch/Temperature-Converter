Celsius, Fahrenheit aur Kelvin ke beech temperature convert karta hai. User input daalta hai aur instant result milta hai. Vanilla JS se DOM manipulation aur formula use kiya hoga.
# Temperature Converter 🌡️
A simple web app to convert temperatures between Celsius, Fahrenheit, and Kelvin.

**Tech Used:** HTML, CSS, JavaScript
**Features:** Real-time conversion, Input validation, Responsive UI
